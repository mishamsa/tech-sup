# Telegram-бот для автоматизації турнірів з настільних ігор

**Бакалаврський проєкт Краснокутського Міхаіла**

## Опис

Бот дозволяє організаторам створювати турніри прямо в Telegram, реєструвати учасників та автоматично генерувати пари для матчів. Система забезпечує зручний доступ до результатів та розкладу ігор у режимі реального часу.

## Функціонал бота

* Реєстрація учасників через інтерфейс Telegram.
* Автоматична генерація турнірних сіток.
* Система адміністрування для створення та завершення івентів.
* Перегляд статистики та результатів ігор.

## Технологічний стек

* **Language:** Python 3.x
* **Framework:** Python-telegram-bot
* **Database:** SQLite
* **Environment:** Docker

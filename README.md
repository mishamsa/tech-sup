# Інформаційна система для автоматизації управління турнірами з настільних ігор

**Бакалаврський проєкт**

## Опис проєкту
Система призначена для автоматизації процесів реєстрації учасників, формування турнірних сіток (Single Elimination, Swiss system) та ведення статистики результатів матчів у настільних іграх.

## Функціонал
- Реєстрація гравців та ведення профілів.
- Автоматична генерація розкладу ігор.
- Розрахунок рейтингів (наприклад, за системою Ело).
- Експорт результатів у формат Excel/PDF.

## Технологічний стек
- **Language:** Python 3.x
- **GUI:** Tkinter
- **Database:** SQLite / PostgreSQL
- **Tools:** Git, SQL

## Як запустити
1. Клонувати репозиторій: `git clone <link>`
2. Встановити залежності: `pip install -r requirements.txt`
3. Запустити головний файл: `python src/main.py`
